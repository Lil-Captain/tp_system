# 基于 https://www.skyinfor.com/index.html 网页的展示及后台管理系统设计

## 1. 项目目标

本项目旨在参考 skyinfor 官方网站，实现一个可以“展示网站信息”与“后台可灵活修改内容”的内容管理系统。前台针对访客展示公司信息，支持首页展示、企业介绍、产品与解决方案、新闻资讯、联系我们等主要模块；后台则提供完善的内容管理与权限控制，方便维护人员进行内容调整。

## 2. 系统功能结构

### 2.1 前台展示模块

- 首页展示：轮播图、公司简介、核心业务概览
- 关于我们：公司介绍、发展历程、团队信息等
- 产品与解决方案：产品列表展示、产品详情页、解决方案分类及内容
- 新闻资讯：分类新闻展示、新闻详情页
- 联系我们：公司联系方式、地图、在线留言表单
- 其它内容模块（如有）：案例、合作伙伴等

### 2.2 后台管理模块

- 登录/权限管理
  - 管理员登录、退出
  - 权限分级（超级管理员/普通编辑）
- 信息内容管理
  - 轮播图管理（增删改查）
  - 公司简介编辑
  - “关于我们”内容管理
  - 产品及解决方案分类、详情维护
  - 新闻及资讯发布与编辑、分类管理
  - 联系方式、地图位置调整
  - 在线留言管理与导出
  - 合作伙伴/客户案例图片与描述管理
- 系统设置
  - 管理员密码修改
  - 站点基本信息配置

## 3. 技术选型

- 后端：PHP（ThinkPHP8 框架）、MySQL
- 前端：HTML5、CSS3、JavaScript（可选VUE或Layui等UI库）
- 其它：验证码、文件上传、富文本编辑、响应式布局支持

## 4. 主要界面设计

- **前台**
  - 顶部导航栏，风格及内容参照 skyinfor 官网
  - 首页：轮播组件、模块化业务区块
  - 其它页面根据需求组件化开发
- **后台**
  - 登录页
  - 主控制台总览（模块入口快捷卡片）
  - 各模块管理列表页与内容编辑表单

## 5. 数据库设计（简要）

主要表：
- 用户表（user）
- 轮播图表（banner）
- 公司信息表（company_info）
- 产品分类与产品表（product_category, product）
- 解决方案表（solution）
- 新闻分类与新闻表（news_category, news）
- 留言表（message）
- 合作伙伴表（partner）

## 6. 权限控制与安全

- 登录鉴权（Session / Cookie）
- 权限分级及路由访问限制
- 表单数据校验及防止XSS、SQL注入

## 7. 部署与运维

- 支持在 PHP8+ 环境下运行
- 推荐 Nginx/Apache+PHP+MySQL 部署方式
- 后台支持数据备份、日志记录

## 8. 其它可扩展点

- 国际化支持
- 移动端自适应
- API接口（为前端或第三方系统提供数据）
